# MissionGrid 快速参考

## 一句话总结

> MissionGrid 是一个可视化无人机任务编排地面站，在 7×9 网格上拖拽设置动作，自动生成可部署的飞行任务包。

---

## 快速启动

```bash
# 安装依赖
pip install PySide6 pymavlink python-tsp numpy

# 启动地面站
cd mission_grid
python app.py
```

- **机头方向**：+Y (B1→B7)

## 操作速查

| 操作 | 方法 |
|------|------|
| 设置动作 | 左键点击格子 → 选择动作 |
| 标记禁飞 | 右键点击格子 |
| 规划路径 | 点击"自动规划" |
| 导出任务 | 点击"导出" |
| 保存方案 | Ctrl+S |
| 切换标签 | Ctrl+1~4 |

---

## 动作类型

| 动作 | 说明 | 参数 |
|------|------|------|
| 起飞 | 起飞点 | - |
| 拍照 | 保存图片 | 保存目录、前缀 |
| 二维码 | 扫描二维码 | 保存目录 |
| YOLO | 动物检测 | 模型路径、置信度 |
| H 降落 | H 点识别降落 | - |
| 降落 | 直接降落 | - |
| 航向 | 设置航向角 | 角度（度） |
| 蜂鸣器 | 响铃 | 音频 ID |
| 舵机 | 开关舵机 | 舵机 ID、状态 |
| 激光 | 开关激光 | 状态、持续时间 |

---

## 触发条件

| 条件 | 说明 |
|------|------|
| 每次经过 | 每次到达都执行 |
| 首次经过 | 仅第一次执行 |
| 最后经过 | 仅最后一次执行 |
| 主线完成后 | 主线任务完成后执行 |

## 部署速查

```bash
# 1. 传输任务包
scp -r mission_grid_YYYYMMDD_HHMMSS orangepi@192.168.144.xxx:/home/orangepi/

# 2. SSH 连接
ssh orangepi@192.168.144.xxx

# 3. 运行任务
cd /home/orangepi/mission_grid_YYYYMMDD_HHMMSS
chmod +x run_mission.sh
./run_mission.sh
```

---

## 降落逻辑

```
有降落动作？
    ├── 是 → 在指定格子降落
    └── 否 → 返回起飞点降落
```

## 常用命令

```bash
# 查看 MAVROS 状态
rostopic echo /mavros/state

# 查看位置
rostopic echo /mavros/local_position/odom

# 查看 SLAM
rostopic echo /Odometry

# 查看节点
rosnode list
```
