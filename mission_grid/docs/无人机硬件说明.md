# 无人机硬件说明

## 概述

本文档记录无人机的硬件配置、接口说明和关键参数，供开发和调试参考。

---

## 1. 硬件配置

### 1.1 核心组件

| 组件 | 型号 | 说明 |
|------|------|------|
| 机载电脑 | OrangePi CM5 | ARM 架构，Ubuntu 20.04 |
| 激光雷达 | Livox MID360 | 3D 点云，用于 SLAM 建图和定位 |
| 下视摄像头 | Intel RealSense | 用于二维码识别、目标检测 |
| 飞控 | PX4 | 通过 MAVROS 通信 |
| 遥控器 | - | 用于手动接管和解锁 |

### 1.2 通信接口

| 接口 | 协议 | 端口/Topic | 说明 |
|------|------|------------|------|
| 飞控通信 | MAVLink | UDP 14555 | MAVROS 连接飞控 |
| 激光雷达 | ROS Topic | /livox/lidar | 点云数据 |
| 摄像头 | ROS Topic | /camera/color/image_raw | 图像数据 |
| SLAM 定位 | ROS Topic | /Odometry | FAST_LIO 输出 |
| 控制指令 | ROS Topic | /mavros/setpoint_raw/local | 位置控制 |

## 2. 坐标系统

### 2.1 机体坐标系

- **X 轴**：机头方向
- **Y 轴**：右侧
- **Z 轴**：下方

### 2.2 MAVROS 坐标系

- **X 轴**：北
- **Y 轴**：东
- **Z 轴**：下（高度为负值）

### 2.3 MissionGrid 坐标系

```
    A1  A2  A3  A4  A5  A6  A7  A8  A9
B7  ·   ·   ·   ·   ·   ·   ·   ·   ·     row=6
B6  ·   ·   ·   ·   ·   ·   ·   ·   ·     row=5
B5  ·   ·   ·   ·   ·   ·   ·   ·   ·     row=4
B4  ·   ·   ·   ·   ·   ·   ·   ·   ·     row=3
B3  ·   ·   ·   ·   ·   ·   ·   ·   ·     row=2
B2  ·   ·   ·   ·   ·   ·   ·   ·   ·     row=1
B1  ·   ·   ·   ·   ·   ·   ·   ·   ★     row=0
    col=0                           col=8
```

- **网格坐标**：(col, row), col=0~8 (A1~A9), row=0~6 (B1~B7)
- **物理坐标**：
  - X = (8 - col) × 0.5 米
  - Y = row × 0.5 米
- **原点**：A9B1 (col=8, row=0) 对应物理坐标 (0, 0)
- **机头方向**：始终朝 +Y (B1→B7)

## 3. 核心接口

### 3.1 uav_ctrl_tools.CtrlTools

```python
import uav_ctrl_tools as ctrl_tools

# 初始化
uav = ctrl_tools.CtrlTools()

# 基础控制
uav.uav_takeoff()          # 起飞
uav.uav_land()             # 降落
uav.set_point(x, y, z)     # 飞到目标坐标
uav.set_yaw(rad)           # 设置航向角（弧度）

# 状态获取
x, y, z = uav.get_local()  # 获取当前位置
uav.init_yaw               # 获取初始航向角

# 外设控制
uav.ctrl_buzzer(id)        # 控制蜂鸣器
uav.ctrl_servo(ch, state)  # 控制舵机
uav.get_img()              # 获取下视摄像头图像
```

### 3.2 uav_img_tools.ImgTools

```python
import uav_img_tools as img_tools

# 初始化
img = img_tools.ImgTools()

# 图像处理
num = img.num(frame)           # CNN 手写数字识别
qr = img.qr_code(frame)       # 二维码检测
circle = img.circle_detect(frame)  # 圆形目标检测
```

## 4. ROS 节点

### 4.1 节点列表

| 节点 | 功能 | 启动命令 |
|------|------|----------|
| MAVROS | 飞控通信 | `roslaunch mavros apm.launch` |
| Livox 驱动 | 激光雷达 | `roslaunch livox_ros_driver2 msg_MID360s.launch` |
| 摄像头驱动 | 图像采集 | `roslaunch cam_pkg cam_pub.launch` |
| SLAM | 定位建图 | `rosrun competition_pkg node_manage.py` |

### 4.2 启动顺序

```bash
# 1. MAVROS（等待 30 秒连接飞控）
roslaunch mavros apm.launch fcu_url:=udp://:14555@192.168.144.15:14550 &
sleep 30

# 2. Livox 激光雷达
roslaunch livox_ros_driver2 msg_MID360s.launch &
sleep 10

# 3. 摄像头
roslaunch cam_pkg cam_pub.launch &
sleep 5

# 4. SLAM（等待定位稳定）
rosrun competition_pkg node_manage.py &
sleep 15
```

## 7. 常见问题

### Q1: MAVROS 启动后无法连接飞控

**检查项**：
1. 飞控是否上电
2. UDP 端口是否正确（14555）
3. 网络连接是否正常

### Q2: SLAM 定位不稳定

**解决方案**：
- 多次重启 `node_manage.py`
- 确保激光雷达数据正常
- 避免在强光/反光环境下使用

### Q3: PyTorch 报错

**解决方案**：
```bash
export LD_PRELOAD=/home/orangepi/.local/lib/python3.8/site-packages/torch/lib/libgomp-d22c30c5.so.1
```
