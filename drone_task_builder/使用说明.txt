无人机任务编排器使用说明

1. 这是一个 Windows 上运行的图形化原型程序。
2. 它的作用是把基础任务组合起来，然后导出成新的机载任务包。
3. 它不会覆盖你原来桌面上的无人机程序。

运行方法：

1. 打开这个目录：
   C:\Users\王震\Documents\无人机\drone_task_builder
2. 如果还没装依赖，先执行：
   pip install -r requirements.txt
3. 运行：
   python app.py
4. 或者直接双击：
   run_app.bat

当前可导出的任务类型：

1. 位置监控任务包
2. 巡航 + 二维码识别/抓拍/坐标计算 + H 识别降落任务包

推荐使用方式：

1. 左侧双击基础任务，加入中间的任务流
2. 右侧修改每个任务的参数
3. 底部填写任务名称和导出目录
4. 点击“导出任务包”

导出后会生成一个新的文件夹，里面一般会包含：

1. 任务描述 json
2. 任务入口 sh
3. 任务入口 py
4. 参考脚本复制出的支持文件

上机使用方法：

1. 把导出的整个任务包文件夹复制到无人机机载电脑
2. 在机载电脑里给 sh 文件加执行权限：
   chmod +x run_generated_mission.sh
   或
   chmod +x run_monitor.sh
3. 然后运行对应的 sh 文件

说明：

1. 这个版本优先复用你之前已经成功运行过的脚本，所以稳定性更高。
2. 现在还不是“任意任务都能自动拼成新代码”的最终版，而是第一版可用原型。
3. 后续还可以继续扩展更多基础任务和更灵活的导出规则。
